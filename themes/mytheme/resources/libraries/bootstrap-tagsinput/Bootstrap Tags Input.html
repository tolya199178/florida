Bootstrap Tags Input

jQuery plugin providing a Twitter Bootstrap user interface for managing tags

Examples

Markup

Just add data-role="tagsinput" to your input field to automatically change it to a tags input field.

Amsterdam Washington Sydney Beijing Cairo london
Show code 
  1. <input type="text" value="Amsterdam,Washington,Sydney,Beijing,Cairo" data-role="tagsinput" />
statementreturns
$("input").val()
"Amsterdam,Washington,Sydney,Beijing,Cairo,london"
$("input").tagsinput('items')
["Amsterdam","Washington","Sydney","Beijing","Cairo","london"]

True multi value

Use a <select multiple /> as your input element for a tags input, to gain true multivalue support. Instead of a comma separated string, the values will be set in an array. Existing <option /> elements will automatically be set as tags. This makes it also possible to create tags containing a comma.

Amsterdam Washington Sydney Beijing Cairo
Show code 
  1. <select multiple data-role="tagsinput">
  2.   <option value="Amsterdam">Amsterdam</option>
  3.   <option value="Washington">Washington</option>
  4.   <option value="Sydney">Sydney</option>
  5.   <option value="Beijing">Beijing</option>
  6.   <option value="Cairo">Cairo</option>
  7. </select>
statementreturns
$("select").val()
["Amsterdam","Washington","Sydney","Beijing","Cairo"]
$("select").tagsinput('items')
["Amsterdam","Washington","Sydney","Beijing","Cairo"]

Typeahead

Typeahead is not included in Bootstrap 3, so you'll have to include your own typeahead library. I'd recommed typeahead.js. An example of using this is shown below.
Amsterdam Washington
Show code 
  1. <input type="text" value="Amsterdam,Washington" data-role="tagsinput" />
  2. <script>
  3. $('input').tagsinput();
  4.  
  5. // Adding custom typeahead support using http://twitter.github.io/typeahead.js
  6. $('input').tagsinput('input').typeahead({
  7.   prefetch: 'citynames.json'
  8. }).bind('typeahead:selected', $.proxy(function (obj, datum) {  
  9.   this.tagsinput('add', datum.value);
  10.   this.tagsinput('input').typeahead('setQuery', '');
  11. }, $('input')));
  12. </script>
statementreturns
$("input").val()
"Amsterdam,Washington"
$("input").tagsinput('items')
["Amsterdam","Washington"]

Objects as tags

Instead of just adding strings as tags, bind objects to your tags. This makes it possible to set id values in your input field's value, instead of just the tag's text.

Amsterdam Washington Sydney Beijing Cairo
Show code 
  1. <input type="text" />
  2. <script>
  3. $('input').tagsinput({
  4.   itemValue: 'value',
  5.   itemText: 'text'
  6. });
  7. $('input').tagsinput('add', { "value": 1 , "text": "Amsterdam"   , "continent": "Europe"    });
  8. $('input').tagsinput('add', { "value": 4 , "text": "Washington"  , "continent": "America"   });
  9. $('input').tagsinput('add', { "value": 7 , "text": "Sydney"      , "continent": "Australia" });
  10. $('input').tagsinput('add', { "value": 10, "text": "Beijing"     , "continent": "Asia"      });
  11. $('input').tagsinput('add', { "value": 13, "text": "Cairo"       , "continent": "Africa"    });
  12.  
  13. $('input').tagsinput('input').typeahead({
  14.   valueKey: 'text',
  15.   prefetch: 'assets/cities.json',
  16.   template: '<p>{{text}}</p>',
  17.   engine: Hogan
  18.  
  19. }).bind('typeahead:selected', $.proxy(function (obj, datum) {
  20. 	this.tagsinput('add', datum);
  21. 	this.tagsinput('input').typeahead('setQuery', '');
  22. }, $('input')));
  23. </script>
statementreturns
$("input").val()
"1,4,7,10,13"
$("input").tagsinput('items')
[{"value":1,"text":"Amsterdam","continent":"Europe"},{"value":4,"text":"Washington","continent":"America"},{"value":7,"text":"Sydney","continent":"Australia"},{"value":10,"text":"Beijing","continent":"Asia"},{"value":13,"text":"Cairo","continent":"Africa"}]

Categorizing tags

You can set a fixed css class for your tags, or determine dynamically by provinding a custom function.

Amsterdam Washington Sydney Beijing Cairo
Show code 
  1. <input type="text" />
  2. <script>
  3. $('input').tagsinput({
  4.   tagClass: function(item) {
  5.     switch (item.continent) {
  6.       case 'Europe'   : return 'label label-primary';
  7.       case 'America'  : return 'label label-danger label-important';
  8.       case 'Australia': return 'label label-success';
  9.       case 'Africa'   : return 'label label-default';
  10.       case 'Asia'     : return 'label label-warning';
  11.     }
  12.   },
  13.   itemValue: 'value',
  14.   itemText: 'text'
  15. });
  16. $('input').tagsinput('add', { "value": 1 , "text": "Amsterdam"   , "continent": "Europe"    });
  17. $('input').tagsinput('add', { "value": 4 , "text": "Washington"  , "continent": "America"   });
  18. $('input').tagsinput('add', { "value": 7 , "text": "Sydney"      , "continent": "Australia" });
  19. $('input').tagsinput('add', { "value": 10, "text": "Beijing"     , "continent": "Asia"      });
  20. $('input').tagsinput('add', { "value": 13, "text": "Cairo"       , "continent": "Africa"    });
  21.  
  22. // Adding custom typeahead support using http://twitter.github.io/typeahead.js
  23. $('input').tagsinput('input').typeahead({
  24.   valueKey: 'text',
  25.   prefetch: 'cities.json',
  26.   template: '<p>{{text}}</p>',                                      
  27.   engine: Hogan
  28. }).bind('typeahead:selected', $.proxy(function (obj, datum) {  
  29.   this.tagsinput('add', datum);
  30.   this.tagsinput('input').typeahead('setQuery', '');
  31. }, $('input')));
  32. </script>

Options

option description
tagClass

Classname for the tags, or a function returning a classname

  1. $('input').tagsinput({
  2.   tagClass: 'big'
  3. });
  1. $('input').tagsinput({
  2.   tagClass: function(item) {
  3.     return (item.length > 10 ? 'big' : 'small');
  4.   }
  5. });
itemValue

When adding objects as tags, itemValue must be set to the name of the property containing the item's value, or a function returning an item's value.

  1. $('input').tagsinput({
  2.     itemValue: 'id'
  3. });
  1. $('input').tagsinput({
  2.   itemValue: function(item) {
  3.     return item.id;
  4.   }
  5. });
itemText

When adding objects as tags, you can set itemText to the name of the property of item to use for a its tag's text. You may also provide a function which returns an item's value. When this options is not set, the value of itemValue will be used. 

  1. $('input').tagsinput({
  2.   itemText: 'label'
  3. });
  1. $('input').tagsinput({
  2.   itemText: function(item) {
  3.     return item.label;
  4.   }
  5. });
confirmKeys

Array of keycodes which will add a tag when typing in the input. (default: [13], which is ENTER)

  1. $('input').tagsinput({
  2.   confirmKeys: [13, 44]
  3. });
maxTags

When set, no more than the given number of tags are allowed to add (default: undefined). When maxTags is reached, a class 'bootstrap-tagsinput-max' is placed on the tagsinput element.

  1. $('input').tagsinput({
  2.   maxTags: 3
  3. });
freeInput

Allow creating tags which are not returned by typeahead's source (default: true)

This is only possible when using string as tags. When itemValue option is set, this option will be ignored. 
  1. $('input').tagsinput({
  2.   typeahead: {
  3.     source: ['Amsterdam', 'Washington', 'Sydney', 'Beijing', 'Cairo'],
  4.     freeInput: true
  5.   }
  6. });
typeahead

Object containing typeahead specific options
source

An array (or function returning a promise or array), which will be used as source for a typeahead. 

  1. $('input').tagsinput({
  2.   typeahead: {
  3.     source: ['Amsterdam', 'Washington', 'Sydney', 'Beijing', 'Cairo']
  4.   }
  5. });
  1. $('input').tagsinput({
  2.   typeahead: {                  
  3.     source: function(query) {
  4.       return $.get('http://someservice.com');
  5.     }
  6.   }
  7. });
onTagExists

Function invoked when trying to add an item which allready exists. By default, the existing tag hides and fades in. 

  1. $('input').tagsinput({
  2.   onTagExists: function(item, $tag) {
  3.     $tag.hide.fadeIn();
  4.   }
  5. });

Methods

method description
add

Adds a tag

  1. $('input').tagsinput('add', 'some tag');
  1. $('input').tagsinput('add', { id: 1, text: 'some tag' });
remove

Removes a tag

  1. $('input').tagsinput('remove', 'some tag');
  1. $('input').tagsinput('remove', { id: 1, text: 'some tag' });
removeAll

Removes all tags

  1. $('input').tagsinput('removeAll');
focus

Sets focus in the tagsinput

  1. $('input').tagsinput('focus');
input

Returns the tagsinput's internal <input />, which is used for adding tags. You could use this to add your own typeahead behaviour for example.

  1. var $elt = $('input').tagsinput('input');
refresh

Refreshes the tags input UI. This might be usefull when you're adding objects as tags. When an object's text changes, you'll have to refresh to update the matching tag's text.

  1. $('input').tagsinput('refresh');
destroy

Removes tagsinput behaviour

  1. $('input').tagsinput('destroy');
